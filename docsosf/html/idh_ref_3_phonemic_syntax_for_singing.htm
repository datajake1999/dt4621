Phonemic Syntax for Singing

Use the following command syntax to code musical sequences:

Phoneme Syntax for Singing

SYNTAX:

[phoneme <duration, pitch number>]

OPTIONS:

none

PARAMETERS:

Duration: Tone duration in milliseconds.

 

Pitch Number: Pitch number from Tone Table

DEFAULT:

none
