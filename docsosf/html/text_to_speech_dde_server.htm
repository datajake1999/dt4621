Text-to-Speech Server (Windows 95/98/ME/NT/2000)

The Text-to-Speech (TTS) server for Windows makes it possible for any Windows application that can call Dynamic Data Exchange (DDE) and that contains ASCII text to speak that text through DECtalk Software.

Under Windows, the server is launched automatically whenever you start Microsoft Mail or manually whenever you click the Text-to-Speech server icon in the DECtalk Software program group.
