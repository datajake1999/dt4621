TextToSpeechVersionEx

The TextToSpeechVersionEx call returns information about the currently running version of DECtalk Software.

Syntax

ULONG TextToSpeechVersionEx

(LPVERSION_INFO *ver)

 

Parameters

LPVERSION_INFO *ver

The address of a pointer to an array of characters with version information. The VERSION_INFO structure is as follows:

 DWORD StructSize;

 DWORD StructVersion;

 WORD DLLVersion;

 WORD DTalkVersion;

 LPSTR VerString;

 LPSTR Language;

 DWORD Features;

 

 

 

Return Value

This call returns an unsigned long integer (ULONG) with the size of the VERSION_INFO structure. The return value is zero if the call is NOT successful. No other error information is available.

 

 

Example

 

BOOL IsDECtalkMLInstalled(void) {

 LPVERSION_INFO verinfo;

 

 TextToSpeechVersionEx(&verinfo);

 if (ver->Features & TTS_FEATS_MULTILANG) return TRUE;

 return FALSE;

}
