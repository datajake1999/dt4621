Loading a User Dictionary from the Speak Applet

Click here to see a numbered graphic illustrating the following procedure.

Task/Location

 

Action

Result

File pull-down menu (3)

Step 1

Select Load User Dictionary. (9) Note that the current User Dictionary is defined in the dectalk.ini file and is loaded if available.

The Load Dictionary dialog box (not shown here) is displayed.

 

Step 2

Select the User Dictionary you want to load, and press Enter or click OK.

The dictionary you select is loaded and remains in effect until you change the dictionary or end the session.
