Editing a Text File from the Dtsample Applet

Click here to see a numbered graphic illustrating the following procedure.

DECtalk Software lets you edit text — select, cut, paste, append other text files — that is displayed in the Dtsample applet dialog and then play segments or all of the edited text.

You can also increase and decrease the rate at which DECtalk Software speaks a file or a section of a file.

To edit a text file:

Task/Location

 

Action

Result

DECtalk Software Dtsample applet dialog (1)

Step 1

Select the range of text you want to edit. (2)

 

 

Step 2

Use cut, copy, paste, and delete selections from the Edit pull-down menu (3) to manipulate the selected text. You can also insert text and DECtalk Software commands by placing the cursor anywhere in the text and typing.

 

 

Step 3

Click the Start button. (4)

DECtalk Software speaks the edited file in the new, edited format.
