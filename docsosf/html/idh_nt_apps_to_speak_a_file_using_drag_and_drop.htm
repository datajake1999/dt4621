Speaking a Text File Using Drag and Drop

Task/Location

 

Action

Result

Windows Explorer with text file name visible (not shown here)

Step 1

Click the file name, and drag it to either:

• the Dtsample applet or the
Speak applet dialog

• the minimized DECtalk
Dtsample or Speak icons in
the program group or on the
desktop.

The file is spoken.

If you drag the file to a DECtalk icon, the application it represents is opened.
