About the Dtsample Applet

The fastest way to start learning about DECtalk Software is by using the Dtsample applet to convert an ASCII text file to speech. Once you start the applet and speak a text file, you can then learn more about controlling voice characteristics through voice-control commands and the user dictionary, which are covered in other Help topics.

Using the DECtalk Software Dtsample applet, you can:

Open a text file for speaking

Insert voice control commands

Change voice selections

Using the speaking voice menu

Using voice control commands

Edit a text file

Play selected text

Change the speaking rate
