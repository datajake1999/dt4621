TextToSpeechTyping

The TextToSpeechTyping call speaks a single letter as quickly as possible, aborting any previously queued speech. This will be somewhat slower if TextToSpeechSpeak has been called since the last TextToSpeechTyping or TextToSpeechReset call.

This call is primarily useful with the Access32 versions of DECtalk Software. This calls exists in non-Access32 versions, but is not particularly fast.

Syntax

void TextToSpeechTyping

(LPTTS_HANDLE_T phTTS, char cLetter)

Parameters

LPTTS_HANDLE_T phTTS

Specifies a text-to-speech handle.

 

char cletter

The letter to speak.

Return Value

None.

 

 

 

Comments

This function should only be called only when the application is synthesizing directly to an audio device (not to memory or to a file).
